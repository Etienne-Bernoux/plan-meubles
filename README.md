# Plan Meuble

Collection de plans de meubles avec vues 3D interactives (Three.js) et plans cotés PDF.

## Utilisation

Disponible en ligne : https://etienne-bernoux.github.io/plan-meubles/

## Meubles disponibles

- **Meuble Cantinière** — vue 3D
- **Rangement Chutes de Bois** — vue 3D + plan coté PDF

## Ajouter un meuble

1. Créer un dossier `nom-du-meuble/`
2. Y ajouter le fichier `*-3d.html` (Three.js)
3. Y ajouter le PDF du plan coté si dispo
4. Mettre à jour `index.html`

Voir `CLAUDE.md` pour les conventions détaillées.
